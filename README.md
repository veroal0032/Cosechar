# Cosechar — Frutas y verduras de estación 🌿

Web app para explorar qué frutas y verduras están en temporada en Argentina, con información nutricional, aliases regionales latinoamericanos y una UI que cambia según la estación del año.

🔗 [cosechar.vercel.app](https://cosechar.vercel.app)

## Funcionalidades

- Calendario estacional para Argentina (4 estaciones)
- Soporte bilingüe español / inglés
- Datos nutricionales por producto
- Aliases regionales latinoamericanos (distintos nombres para el mismo producto según el país)
- UI adaptiva: colores y estética cambian según la temporada activa
- Analytics con Vercel Analytics

## Correr localmente

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Estructura del proyecto

```
app/          → páginas y layouts de Next.js
data/         → datos de productos por estación
public/       → assets estáticos
```
